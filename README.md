script em bash feito com intuido de automatizar a consulta de subdomínios com o auxílio das ferramentas
-> Curl  ; gobuster ; subfinder ; httpx-toolkit e wafw00f.
Desenvolvido por:
-> Fábio Assis
Observação .: Não use do script para prejudicar terceiros!

Lembrando que para rodar temos que ter em seu sistema linux os seguintes programas abaixo:
-> zenity
-> yad
-> curl
-> gobuster
-> subfinder
-> httpx-toolkit
-> wafw00f

USO:
Baixe o arquivo severinogo.sh para dentro do diretório de sua preferencia
abra o terminal e digite os seguintes comandos abaixo:
1- bash
2- chmod +x severinogo.sh
3- bash severinogo.sh
